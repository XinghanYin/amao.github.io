代码

无论你的母语是日语，印地语还是加泰罗尼亚语，源代码已经成为今天的通用语言。Ruby，Objective-C, jQuery——这些语言是现代知识的无形支柱。它们影响我们的判断并驱动我们的决策，并且，通过使开发者保持常备不懈，它们的不断演变推动技术向前发展，下一波进展将会引领我们去何方？没有任何线索。但是，最低程度的代码能力正在变得越来越重要，即使不是必需的。
